# Owned Reactivation

Status: proposed owned-list campaign
Target first batch: 10 contacts after suppression review
Primary learning intent: fast learning from prior intent

## Persona

Existing Julep or Memory Store power users, triers, enriched leads, prior demo conversations, and warm accounts.

## Pain

The prospect already showed intent or product familiarity. The campaign tests whether the new shared-agent-memory surface solves a current workflow.

## Hard Criteria

- Contact exists in an approved owned source.
- Suppression, unsubscribe, active-thread, duplicate, and do-not-contact checks pass.
- Prior interaction can be referenced without overclaiming or leaking private context.

## Trigger Signals

- Prior signup, usage, demo, revenue, GitHub activity, or direct product interaction.
- Recent Memory Store or Julep-adjacent thread.
- Known workflow where shared memory could matter now.

## Proof Path

Reference prior usage carefully, then point to the current Memory Store surface: one shared context layer for the team's agents.

## Exclusions

- Unsubscribed or do-not-contact contacts.
- Negative replies.
- Existing active sales/customer conversations unless owner approves.
- Contacts where prior context is too thin to reference safely.

## Source Plan

This ICP is not an Exa/Websets public-sourcing pool. Use an approved owned-list import plus Gmail/suppression review.
